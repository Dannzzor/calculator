# Calculator

Learning Elixir, building out a calculator app with Elixir and Phoenix LiveView.

## Installation

1. Install dependencies:

```bash
mix deps.get
```

## Running the Application

1. Start the server:

```bash
mix run --no-halt
```

2. Open your browser and navigate to:

```
http://localhost:4000
```

3. Kill the process

```
pkill -f "mix run" || true
```

## Technology Stack

- **Elixir**: Functional programming language
- **Phoenix LiveView**: Real-time, interactive web applications
- **Plug Cowboy**: HTTP server
